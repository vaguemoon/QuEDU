# 專案檔案功能說明

## 檔案結構與功能

### curriculum.js
- **功能**：負責課程選擇與自由練習的資料處理。
- **主要職責**：
  - 從 Firestore 載入課程資料。
  - 實現三層下拉選單的功能。
  - 提供課程啟動相關函數，例如 `startCurriculumLesson()`。
- **依賴檔案**：`state.js`、`nav.js`、`menu.js`。

### dictation.js
- **功能**：實現默寫模式（自由手寫）。
- **主要職責**：
  - 切換到默寫模式。
  - 初始化手寫 Canvas。
  - 提交默寫結果與自評功能。
- **依賴檔案**：`state.js`、`menu.js`、`shared.js`。

### exam.js
- **功能**：負責「今日測驗」的多輪測驗邏輯。
- **主要職責**：
  - 啟動完整測驗流程。
  - 顯示測驗題目、播放語音、記錄結果。
  - 提供多輪測驗的總結與最終結果顯示。
- **依賴檔案**：`state.js`、`nav.js`、`voice.js`、`shared.js`。

### init.js
- **功能**：應用程式啟動與自動登入。
- **主要職責**：
  - 初始化 Firebase、主題與音效。
  - 自動從 `sessionStorage` 登入。
  - 載入課程資料。
- **依賴檔案**：`shared.js` 及所有其他模組。

### menu.js
- **功能**：管理生字選單與學習進度條。
- **主要職責**：
  - 渲染選單。
  - 更新進度條。
  - 處理選單卡片點擊事件。
- **依賴檔案**：`state.js`、`nav.js`、`voice.js`、`practice.js`。

### nav.js
- **功能**：負責畫面切換與導航堆疊。
- **主要職責**：
  - 提供頁面切換功能。
  - 管理導航堆疊（`PAGE_STACK`）。
  - 與外層框架進行訊息通信。

### practice.js
- **功能**：筆順練習模式。
- **主要職責**：
  - 切換到練習模式。
  - 初始化與重啟測驗。
  - 提供單字測驗與結果顯示功能。
- **依賴檔案**：`state.js`、`nav.js`、`menu.js`、`shared.js`。

### state.js
- **功能**：全域狀態管理與 Firebase 資料存取。
- **主要職責**：
  - 管理學生與學習狀態。
  - 提供進度與筆畫的儲存功能。

### voice.js
- **功能**：語音朗讀功能。
- **主要職責**：
  - 載入中文語音。
  - 使用 Web Speech API 朗讀單字。
- **依賴檔案**：`shared.js` 的音效設定。
