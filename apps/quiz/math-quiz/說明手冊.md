## 功能概覽

「四則運算測驗」讓教師以**規則參數**即時生成題目，無需預先建立題庫。

- **加、減、乘、除**可混合出題，或單獨測驗任一運算
- 每次學生進入測驗時，系統依教師設定的規則**即時隨機生成**一組全新題目
- 系統自動批改，並為每道錯題標注**錯誤類型**（進位錯誤、退位錯誤、乘法表小錯⋯）
- 教師可在後台查看班級整體答題情況與錯誤分布

## 建立測驗

1. 登入教師後台，點擊左側 **🔢 數學測驗** 分頁
2. 點擊右上角 **＋ 建立測驗** 按鈕
3. 填寫表單：

| 欄位 | 說明 |
|------|------|
| 測驗名稱 | 學生在列表中看到的名稱，例如「第三單元加減法練習」 |
| 總題數 | 5 至 20 題，以滑桿調整 |
| 運算類型 | 勾選想要出的運算（至少勾選一種）；勾選後展開詳細設定 |

4. 設定完成後點 **建立**，測驗即儲存並出現在列表中

> **提示**：若同時勾選多種運算，題目會依比例均勻混合。例如勾選加法與減法、共 10 題，則各出 5 題。

## 運算參數說明

### 加法

| 參數 | 說明 |
|------|------|
| 被加數位數 | 第一個數字的位數（1 = 個位，2 = 十位以下，以此類推）|
| 加數位數 | 第二個數字的位數 |
| 進位模式 | 見下方說明 |

**進位模式：**
- **不進位**：每個位數相加皆小於 10，適合初學練習
- **需進位**：至少有一個位數產生進位
- **連續進位**：至少有兩個連續位數都進位，難度較高

### 減法

| 參數 | 說明 |
|------|------|
| 被減數位數 | 第一個數字的位數 |
| 減數位數 | 第二個數字的位數（自動確保結果 ≥ 0）|
| 退位模式 | 不退位 / 需退位 / 連續退位（同加法邏輯）|

### 乘法

| 參數 | 說明 |
|------|------|
| 被乘數位數 | 第一個數字的位數 |
| 乘數位數 | 第二個數字的位數 |
| 尾數有 0 | 勾選後，被乘數個位為 0（如 30 × 4）|
| 積中間有 0 | 勾選後，乘積中間某位數字為 0（如 102、3006），難度較高 |

> **注意**：「積中間有 0」使用隨機重試取樣，若位數設定太小（如 1 位乘 1 位）難以達成，系統最多重試 150 次後改用備用題目。

### 除法

| 參數 | 說明 |
|------|------|
| 除數位數 | 除數的位數 |
| 商的位數 | 商（答案）的位數 |
| 結果類型 | **整除**：無餘數；**有餘數**：學生需同時填入商與餘數 |
| 商尾數有 0 | 勾選後，商的個位為 0（如商 = 30）|
| 商中間有 0 | 勾選後，商的中間某位數字為 0（如商 = 102）|

> **注意**：「有餘數」時，餘數恆小於除數。學生作答時會看到**餘數大於除數**的警告提示。

## 派發測驗給學生

1. 在數學測驗列表中，找到要派發的測驗，點擊 **📤 派發**
2. 在彈出視窗中勾選一或多個班級
3. 點擊 **儲存設定**

派發完成後，學生進入 Hub 的「測驗區」即可看到測驗卡片。

> **注意**：被停用（狀態為「已關閉」）的測驗不會顯示在學生端。

## 查閱學生結果

1. 在列表中點擊 **📊 結果**
2. 結果視窗顯示：
   - 每位學生的得分與正確題數
   - 每道題的答題狀況與錯誤標籤
   - 底部統計：**各錯誤類型在班級中出現的次數**（例如「進位錯誤 × 8」表示有 8 人答錯且被分類為進位問題）

### 如何運用錯誤統計

| 錯誤類型多 | 教學建議 |
|-----------|---------|
| 進位錯誤 / 退位錯誤 | 重新練習進退位規則；可調低難度先從不進位開始 |
| 乘法表小錯 | 安排乘法表熟記練習 |
| 部分積錯誤 | 加強「分拆乘法」教學（先算個位，再算十位）|
| 商計算錯誤 | 加強除法直式練習 |
| 餘數計算錯誤 | 加強「被除數 − 除數 × 商 = 餘數」的步驟練習 |

## 停用與重新啟用測驗

- 在列表「操作」欄中點 **停用**：測驗立即從學生端隱藏，但歷史結果不受影響
- 點 **啟用**：測驗重新對學生開放

適合在測驗期間（如月考前）關閉非必要項目，或測驗結束後收回。

## 學生作答流程

1. 學生進入 Hub，在「測驗區」點擊數學測驗卡片
2. 看到可進行的測驗列表，點擊後**即時生成**一組題目
3. 作答介面：
   - 上方顯示算式（直式版面供參考）
   - 答案欄分割為**一格一格的數字格**，格數等於正確答案的位數
   - 點選一個格子 → 格子反藍（選中）→ 用螢幕鍵盤輸入數字
   - 輸入後自動跳至下一格；按 **⌫** 清除當前格並回退一格
   - 除法有餘數時，需分別填入「商」與「餘數」
4. 底部進度點可隨時跳題；未填的題目仍可送出（計為答錯）
5. 到達最後一題後，按 **送出答案** 查看結果
6. 結果頁顯示每題對錯、你的作答、錯誤標籤，以及總分

## 錯誤標籤對照表

| 標籤 | 運算 | 意義 |
|------|------|------|
| 進位錯誤 | 加法 | 答案與正解相差恰好為 10 的冪次（如差 10、100），推斷為忘記進位 |
| 退位錯誤 | 減法 | 答案與正解相差恰好為 10 的冪次，推斷為忘記退位 |
| 計算錯誤 | 加/減 | 差值不符合進/退位模式，屬一般計算失誤 |
| 乘法表小錯 | 乘法 | 答案與正解的差 ≤ 較大的因數，推斷為乘法口訣記錯一位 |
| 部分積錯誤 | 乘法 | 答案近似於只算了個位或只算了十位的部分積，推斷為遺漏了一步 |
| 計算錯誤 | 乘法 | 不符合上述規律的乘法失誤 |
| 餘數大於除數 | 除法 | 學生填入的餘數 ≥ 除數（邏輯上不可能，學生作答時也會即時提醒）|
| 商計算錯誤 | 除法 | 商錯但餘數正確 |
| 餘數計算錯誤 | 除法 | 商正確但餘數錯 |
| 計算錯誤 | 除法 | 商與餘數皆錯 |

> **說明**：所有錯誤標籤均為**推斷式**，依最終答案與正解的差值自動判定，不代表百分之百確定學生的錯誤原因，供教師參考。

## 常見問題

**Q：每次學生開啟同一測驗，題目都一樣嗎？**
不一樣。題目在學生端即時隨機生成，每次開啟都是一組新題目，符合教師設定的規則。

**Q：學生沒有作答的題目如何計分？**
空白格視為答錯，不影響其他題目的計分。

**Q：可以讓多個班級同時進行同一個測驗嗎？**
可以。在「派發」時勾選多個班級即可。

**Q：測驗關閉後，已提交的結果還在嗎？**
是的，停用測驗只是隱藏給學生，不刪除任何已提交的結果。教師仍可在「📊 結果」中查看。

**Q：除法設為「整除」，學生看到的介面有餘數格嗎？**
沒有。整除題目只顯示商的輸入格；有餘數的題目才會出現「餘數」欄。
